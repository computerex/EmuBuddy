## Platform-Specific Notes

### Windows
- All emulators available as portable archives
- No installation required
- Extract and run

### Linux
- **Flatpak:** Modern distro-agnostic package format (Dolphin, PPSSPP)
- **AppImage:** Portable executable format (PCSX2, mGBA)
- **Snap:** Ubuntu/Snap-enabled distros (DeSmuME)
- **Note:** Azahar not available - use alternative or compile from source

### macOS
- All available as DMG installers or archives
- Azahar has 3 versions: Universal (recommended), ARM64, Intel
- RetroArch is Metal-optimized for better performance
- Dolphin is Universal binary (works on both Apple Silicon and Intel)

## Installation Methods by Platform

### Windows
```bash
# Extract with 7-Zip
7za x emulator.7z
```

### Linux (Flatpak)
```bash
flatpak install <flatpak-file>
```

### Linux (AppImage)
```bash
chmod +x emulator.AppImage
./emulator.AppImage
```

### macOS
```bash
# Mount DMG and drag to Applications
open emulator.dmg
```
